# MCRAW Studio

**Render your MotionCam Pro RAW into cinema-grade video. Free.**

A Windows desktop transcoder for `.mcraw` footage. True ACES colour pipeline, all the codecs you actually use, drag-drop batch rendering. No subscription, no plug-in trail, no telemetry.

## What it does

- **Codecs.** ProRes 422 / HQ / 4444 / XQ, DNxHR HQX / 444, CineForm, H.264, H.265 (CPU + NVENC), AV1 NVENC, EXR (PIZ / ZIP / DWA).
- **Colour.** OCIO studio-config-v4.0.0. ACEScg, ACEScct, ACES2065-1, S-Log3, Rec.709 / 2020 / PQ / HLG, DaVinci WG, ADX10.
- **Highlight recovery.** Kills the magenta-blob artifact on blown suns.
- **Lens shading + denoise.** Per-frame LSM, edge-preserving chroma/luma NR.
- **MCRAW trim.** Bit-perfect lossless cuts, no transcode.
- Multi-clip parallel rendering, pause / resume / cancel, JSON presets.

## What you get

- Single 62 MB Windows .exe. No installer, no admin, no service.
- Win 10 / 11 x64.
- Every release is published with a SHA-256 — verify against [VERSIONS.md](VERSIONS.md) or the matching [GitHub Release](https://github.com/outoftokyo/MCRAW_Studio/releases) before running.
